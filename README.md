#  Smart Hospital Appointment & Bed Management System

##  Overview

A web-based application developed to manage hospital operations such as patient registration, appointment scheduling, and bed allocation efficiently. The system reduces manual work and improves hospital resource management.

---

##  Features

*  Admin Login (JWT Authentication)
*  Add & Manage Patients
*  Book & View Appointments
*  Add, Assign & Discharge Beds
*  Data stored in MongoDB

---

##  Tech Stack

**Frontend:** React.js, Axios
**Backend:** Node.js, Express.js
**Database:** MongoDB

---

##  How to Run

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs on: `http://localhost:5000`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on: `http://localhost:5173`
